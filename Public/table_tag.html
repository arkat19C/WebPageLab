This table tag allows users to put a table on the webpage. For example, the code written puts the table of heights and dates on the top of the screen.

<table>

The tag above is the initial tag needed for the table. Below, are the other tags necessary for making a table.

<tr> <th> <td>

"tr" defines a table row, "th" defines a table header, and "td" defines a table cell.

There are many other options for the table, such as allignment, border, and size, all of which are available on the HTML Table Tag website.
