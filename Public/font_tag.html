Using CSS font tags, you can change how text looks on a webpage.

font-family: sans-serif, fantasy;

The code above uses "font-family" to change the font type of the text. Sans-serif and fantasy are both types of font.

font-style: italic;

The code above makes texts italic, such as this line.
